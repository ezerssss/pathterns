# PathTerns

Education game about traversal algorithms. Follow a pattern using either DFS or BFS.

🏷️ Status: Archived \
🤖 Vibe Coded: No

---

## What it does

- Fast simple game where you traverse a predefined pattern using either DFS or BFS.
- Leaderboard ranking based on how fast you complete a pattern and what type of traversal.
